2. IKE Crypto Profile

3. IPSec Crypto Profile

Name: ipsec-crypto-profile

ESP Encryption: aes-128-gcm

Authentication: none

DH Group: group20

---------------------------------------------

4. IKE Gateway
Go to: Network > IKE Gateways

Name: ike-gateway

Version: IKEv2 only

Interface: ethernet1/1

Local IP: 2.2.2.2

Peer IP: 1.1.1.1

IKE Crypto Profile: ike-crypto-profile

5. IPSec Tunnel
